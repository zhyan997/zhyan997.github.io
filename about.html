Data Publication

This course introduces software techniques and tools for building Web browser apps written in HTML5, CSS, JavaScript, and Python with emphasis on user interfaces for presenting information to researchers and students in the humanities. Students learn to create content and to evaluate various approaches for their applicability to humanities research data.
